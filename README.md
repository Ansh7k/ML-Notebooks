## 🚀 Flagship Project — GroundTruth

> A production-grade **RAG (Retrieval-Augmented Generation) pipeline** over legal contracts.

- **Dataset:** CUAD (Contract Understanding Atticus Dataset) — 500+ legal contracts with pre-annotated Q&A pairs
- **Stack:** LangChain · ChromaDB · OpenAI / HuggingFace Embeddings · RAGAS
- **Evaluation:** Faithfulness · Answer Relevancy · Hallucination Rate (via RAGAS)
- **Why it matters:** Automated RAG evaluation is rare in student portfolios — this is the differentiator.
